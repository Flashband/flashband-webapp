flashband-web
=========

Install 
==========

    $ git clone git@github.com:Flashband/flashband-webapp.git

    $ cd flashband-webapp/api/

    $ npm install
    
    $ bower install

Run Tests
==========

* for unit tests:

  * $ gulp test

    or

* for e2e test:

  * $ gulp protractor

Run Code Analysis
=================
    $ gulp scripts


Start Server
=========
    $ gulp serve
